# Basic Go Project
This is a basic Go project for learning purposes some of the features are:
- [x] Basic Go Project Structure
- [x] REST API
- [x] Dockerfile
- [x] Docker Compose
- [x] Fiber
- [x] MongoDB

## How to run
### Docker
```bash
docker-compose up
```
### Local
```bash
go run main.go
```

## How to test
### Docker
```bash
docker-compose exec app go test ./...
```

### Local
```bash
go test ./...
```
